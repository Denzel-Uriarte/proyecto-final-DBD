## Contexto

Sistema de base de datos para el **Hotel Alpheus**, un hotel de 5 estrellas en Tepic, Nayarit. El esquema modela inventario de habitaciones (≥6 categorías), huéspedes, reservaciones y cancelaciones, servicios internos y externos, paquetes promocionales por temporada, facturación, control de quejas y satisfacción del cliente.
